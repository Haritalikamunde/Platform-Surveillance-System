Description :
Developed an automated system monitoring and surveillance tool using Python to track system performance and process activities.

Features :
•	Implemented real-time logging of CPU usage, RAM utilization, disk statistics, network activity, and running processes using the psutil library.
•	Integrated scheduling automation for periodic log generation and system monitoring.
•	Designed a command-line based monitoring utility with detailed reporting for system analysis and administrative monitoring.

Tech Stack: Python, psutil, OS Module, schedule, Automation Scripting
